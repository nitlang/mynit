# NAME

nitserial - generates a serialization support module

# SYNOPSIS

nitserial [*options*]... FILE

# OPTIONS

### `-o`, `--output`
Output file (can also be 'stdout').

### `--dir`
Output directory.

### `-d`, `--depth`
Depth of the visit and generation <module, group, package>.
